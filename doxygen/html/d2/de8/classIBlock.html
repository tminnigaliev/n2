Производит симуляцию: если блок сложный, то симуляция по входящим в него блокам происходит автоматически Если простой, то этот метод переопределён. Если метод не переопределён, то получим исключение.

Аргументы
[in]tмодельное время, для которого расчитываем результат симуляции
[in]ttlпараметр, позволяющий избежать зацикливания, убавляется при каждой новой рекурсии
[in]portвыходной порт для которого надо произвести симуляцию
[in]whereстрока, которая позволяет локализовать ошибку во время симуляции
[in]indexиндекс на случай, если выходной порт векторный

Замещается в TBlock, TWriterToOctMatrix, TMultiplierBlock, TReaderFromOctMatrix, TKohonen и TConstBlock.
