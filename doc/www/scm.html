SCM (software configuration management) проекта N2
В репозитарии есть 3 ветки:
Название ветви Назначение
master хранит скрипты для выкачивания кода и онлайн-документации
code хранит код на языке С++ и конф.файлы и исходники для Doxygen-a
gh-pages хранит онлайн документацию, сгенерированную Doxygen-ом и вэб-странички написанные от руки
